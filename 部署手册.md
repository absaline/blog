# 部署手册：AI辅助零基础搭个人博客

## 方式一：Vercel（推荐）

1. 打开 https://vercel.com/new
2. Import 仓库 `absAline/blog`
3. 框架自动识别为 Astro
4. 环境变量：
   - `PUBLIC_SITE_URL` = 你的域名
   - `GISCUS_REPO` = `你的用户名/仓库名`
   - `GISCUS_REPO_ID` = （见下方说明）
   - `GISCUS_CATEGORY` = `Announcements`
   - `GISCUS_CATEGORY_ID` = （见下方说明）
5. 点击 Deploy

### Giscus 配置
1. 在 GitHub 仓库 Settings → Discussions 启用 Discussions
2. 打开 https://giscus.app，选择你的仓库
3. 获取 repo-id 和 category-id
4. 填入 Vercel 环境变量

## 方式二：Netlify

1. 打开 https://app.netlify.com/start
2. 连接 GitHub 仓库
3. Build command: `npm run build`
4. Publish directory: `dist`
5. 部署完成

## 方式三：本地构建

```bash
cd blog
npm install
npm run build     # 构建到 dist/
npm run preview   # 本地预览构建结果
```

## 后台编辑器部署

在 Vercel 中单独部署 `admin/` 目录，或使用本地 Node.js 运行：

```bash
cd admin
npm install
node server.js   # 端口 4322
```
